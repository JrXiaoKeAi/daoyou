# CSS 规范 (万界道友)

## 1. 核心原则
- **实用性优先**: 遵循 Tailwind CSS 的 Utility-First 理念。
- **响应式设计**: 始终采用移动端优先（Mobile-First）的方法。
- **一致性**: 通过定义配置（Tailwind Config）确保间距、颜色、排版的全局一致性。

## 2. 工具类使用
- 避免在 HTML 中堆砌过长的工具类字符串。对于复杂的重复模式，使用 `@apply` 在 CSS 模块中进行抽象。

## 3. 水墨意境设计指南
- **配色**: 优先使用水墨灰、浓墨黑、宣纸白等传统水墨色调。
- **间距**: 采用适度的留白，体现中式排版的呼吸感。
- **交互**: 提供轻量级的平滑过渡和反馈效果。

## 4. 动画与性能
- 避免使用影响性能的重绘属性。
- 优先使用合成层属性（`transform`, `opacity`）进行动画。
