# 万界道友 (daoyou.org) - 技术栈

## 1. 核心应用架构 (Core Application Architecture)
- **应用框架**: Next.js 16 (App Router)
- **渲染框架**: React 19
- **编程语言**: TypeScript 5
- **样式系统**: Tailwind CSS 4 + PostCSS

## 2. 数据与存储 (Data and Storage)
- **关系数据库**: Supabase PostgreSQL
- **对象关系映射 (ORM)**: Drizzle ORM
- **缓存与队列**: Upstash Redis (Redis REST API)

## 3. 人工智能 (AI Integration)
- **AI 框架**: Vercel AI SDK (ai)
- **AI 适配器**: @ai-sdk/deepseek, @ai-sdk/openai, @ai-sdk/react

## 4. 基础设施与部署 (Infrastructure and Deployment)
- **部署平台**: Vercel, Cloudflare Workers (OpenNext)
- **容器与执行环境**: Node.js 18+, Edge Runtime
- **CI/CD**: GitHub Actions / Vercel 自动构建

## 5. 开发工具 (Development Tools)
- **构建工具**: Next.js, OpenNext
- **测试框架**: Jest, Supertest
- **代码规范**: ESLint 9, Prettier
- **数据库迁移**: Drizzle Kit
